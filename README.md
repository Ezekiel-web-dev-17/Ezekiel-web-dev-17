- 🔭 **I'm currently working on...**
  - [Domino Clone Lite]: A full-stack backend-focused project that combines online shopping with real-time order tracking, inspired by how Domino’s shows you your pizza’s journey. Users can browse products, add to cart, pay online, and track deliveries live. Vendors can manage their own products and view sales analytics. Drivers update order status (“Picked up → On the way → Delivered”). Admins oversee everything: vendors, drivers, customers, and platform health.

#Key Features

*Authentication & Role-based Access (Admin, Vendor, Customer, Driver).
*Secure Payments with Stripe + webhook confirmations.
*Order Tracking in Real-Time via Socket.IO or Server-Sent Events.
*Background Jobs & Notifications (queue-based emails, SMS updates).
*Analytics Dashboard (MongoDB aggregations for sales, delivery times).
*Error Logging, Monitoring & Testing to make it production-ready.
*CI/CD Deployment (GitHub Actions + Render/Railway, optionally Docker).
